# 📚 Команды

## 👤 Профиль

`!profile`
Показать профиль.

`!setbio <текст>`
Изменить девиз.

Пример:
`!setbio Люблю программирование`

`!setstatus <текст>`
Изменить статус.

Пример:
`!setstatus Онлайн`

---

## 💰 Экономика

`!balance`
Показать баланс.

`!daily`
Получить ежедневную награду.

`!farm`
Собрать урожай и заработать монеты.

`!work`
Выполнить работу и получить награду.

`!crime`
Попытаться заработать деньги с риском штрафа.

`!casino <ставка>`
Сыграть в казино.

Пример:
`!casino 100`

`!pay @пользователь <сумма>`
Передать монеты другому пользователю.

Пример:
`!pay @User 500`

`!top`
Показать рейтинг самых богатых пользователей.

---

## 🛡️ Модерация

`!kick @пользователь`
Исключить пользователя с сервера.

`!ban @пользователь`
Заблокировать пользователя на сервере.

---

## ⚙️ Система

`!ping`
Показать задержку бота.

`!help`
Открыть справку по командам.
